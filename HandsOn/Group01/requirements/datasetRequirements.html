  • Data are in the smart cities domain

    • Los datos abiertos pertenecen al dominio de las cuidades inteligentes. En concreto, a la ciudad de San Francisco (EEUU).

  • Data are available as a CSV file

    • Los datasheet que utilizamos en nuestra práctica están en formato csv para facilitar el desarrollo

  • Data have an open license so it can be published

    • Todos los datos de esta página web tiene licencia abierta para ser publicados

    La mayor parte de los datasheet tienen la licencia "Creative Commons 1.0 Universal"

    link para consultar licencias de datasheet:

  • Data can be easily linked with generic real-world entities

    • Lincamos los datos a través de la localización (nombre de la calle). Como en dbpedia no aparecen muchas calles de San Francisco, lincamos también determinados parkings con edificios en esas localizaciones para poder realizar más consultas
